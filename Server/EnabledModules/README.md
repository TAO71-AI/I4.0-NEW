# I4.0 modules

Place all of the enabled modules here.

## About official modules

Official modules are I4.0 modules that are aproved or have been created by TAO71.

## Downloading modules

### Official modules

To download all of the official modules, run the `download_official_modules.sh` script.

### Non-official modules

> [!WARNING]
> When downloading a non-official module, make sure you TRUST the authors and, if possible, check the module's code to make sure it's not malware.

To download a non-official module, run `git clone [MODULE_REPOSITORY]`.

## Updating modules

To update all of the modules (both official and non-official), run the `update_modules.sh` script.
