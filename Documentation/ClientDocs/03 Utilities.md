# Utilities

Utilities are scripts that might be useful, but not used in the client API by default.

> [!NOTE]
> Utilities are not included in the core API package anymore. To install them, please install the package `I4_0-Client-Utils`.

|Name|What it does|
|----|------------|
|chatbot_tools|Sets some tools for the chatbot. Also handles the processing of tools.|
|format_conversion|Converts formats, for example: **HTML => Markdown**.|
|internet|Internet utilities, scraping, and more.|
