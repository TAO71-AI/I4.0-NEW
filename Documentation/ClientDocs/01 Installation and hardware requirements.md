# Hardware requirements

Keep in mind that these requirements may change in the future.

## Minimum hardware requirements

- OS: Any OS
- CPU: Probably any architecture, 1 core
- RAM: ~64 MB
- GPU: No GPU required
- Disk: 100 MB
- Python: 3.11 or newer

## Recommended hardware requirements

- OS: Any OS
- CPU: Probably any architecture, 2 or more cores
- RAM: 1 GB or more
- GPU: No GPU required
- Disk: 1 GB
- Python: 3.11 or newer

# Installing Python

Follow the guide and documentation of your OS to install Python.

# Creating a Python VENV

It is recommended to run the client using a VENV. To create a VENV, run the following command:
```bash
python -m venv .env
```

In some Operating Systems, this command may change.

# Client installation

You can use Python PIP to install the client:
```bash
pip install "I4_0-Client-PY>=17.0.0"
```
